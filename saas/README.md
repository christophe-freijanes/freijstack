# Applications SaaS Démos

Ce répertoire contient des exemples d'applications SaaS développées pour démontrer mes compétences en DevSecOps, développement backend et frontend, conteneurisation et déploiement.

Chaque sous-répertoire (`app1`, `app2`, etc.) représente une application SaaS distincte avec sa propre architecture, ses technologies et son cycle de vie de déploiement.

## App1: Gestionnaire de Tâches Sécurisé

Une application de gestion de tâches complète avec:
- Authentification et autorisation des utilisateurs.
- API RESTful sécurisée.
- Base de données persistante.
- Interface utilisateur interactive.
- Conteneurisation avec Docker.
- Déploiement potentiellement avec Kubernetes ou un service de conteneurs managé.

## App2: Service de Notification en Temps Réel

Un microservice de notification en temps réel démontrant:
- Utilisation de WebSockets pour la communication en direct.
- Intégration avec une base de données NoSQL.
- Architecture microservices.
- Sécurité par conception pour la transmission de données.
- Déploiement automatisé via CI/CD.

Pour plus de détails sur chaque application, veuillez consulter le `README.md` spécifique dans son répertoire respectif.
